## Quick Deployment

To deploy this solution in Azure Cloud Shell, run:

```powershell
Invoke-WebRequest -Uri "https://raw.githubusercontent.com/DataGuys/CentralThreatIntelligenceV2/main/deploy.ps1" -OutFile "./deploy.ps1"; ./deploy.ps1
```
